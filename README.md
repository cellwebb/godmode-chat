# godmode

Get responses from multiple LLMs at once to find the best answers to your questions

## Installation

```bash
python3.11 -m venv .venv
source .venv/bin/activate
pip3 install --upgrade pip
pip3 install -e ".[dev]"
```

## Run the app

```bash
python3 -m streamlit run app.py
```

### Run tests

```bash
python3 -m pytest
```
